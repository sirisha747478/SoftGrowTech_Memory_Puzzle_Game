# 🤖 AI-Powered Web Application

A modern web application built with React and Vite that integrates AI capabilities using an external API. The app provides intelligent responses and a smooth, interactive user experience.

---

## 🚀 Features

* 🤖 AI-powered functionality (API-based responses)
* ⚡ Fast performance using Vite
* 🎨 Clean and responsive UI
* 🔐 Secure API key handling using environment variables
* 📱 Fully responsive design

---

## 🛠️ Tech Stack

* **Frontend:** React
* **Build Tool:** Vite
* **Styling:** Tailwind CSS
* **API Integration:** External AI API (Gemini)

---

## 📦 Installation & Setup

### 1. Clone the repository

```bash
git clone https://github.com/sirisha747478/SoftGrowTech_Memory_Puzzle.git
cd SoftGrowTech_Memory_Puzzle
```

---

### 2. Install dependencies

```bash
npm install
```

---

### 3. Setup environment variables

Create a `.env.local` file in the root directory and add:

```env
GEMINI_API_KEY=your_api_key_here
```

---

### 4. Run the development server

```bash
npm run dev
```

---

### 5. Open in browser

```
http://localhost:3000
```
